# Frontend (React + Vite)

Minimal frontend para el SaaS: login y dashboard que consume la API en `http://localhost:4000`.

Run locally:

```bash
cd frontend
npm install
npm run dev
```

Notas:
- Usa el usuario seed `admin@acme.test` con contraseña `admin`.
- Asegúrate de que el backend esté corriendo en `http://localhost:4000`.
